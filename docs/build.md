# Build & Test

## Build

Build Server

* `./gradlew copyJar` - Build server project.

Build Client

* `npm install` - Install related packages.
* `vsce package` - Build server project.
* `mkdocs -h` - Print help message and exit.

## Test

Test Server

* `./gradlew test` - Test server.

Test Client

* `npm run test` - Test client
